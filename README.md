# CRM: Экспорт активностей

Приложение для Bitrix24 — выгрузка активностей по лидам, сделкам, контактам и компаниям в Excel.

## Функциональность

- Фильтрация по периоду создания
- Фильтрация по типу сущности (Лид, Сделка, Контакт, Компания)
- Фильтрация по статусу (Завершено / Активно)
- Фильтрация по типу активности (Звонок, Email, Задача, Встреча, Чат)
- Предпросмотр данных в таблице
- Выгрузка в Excel (.xlsx)

## Технологии

- VibeCode API — получение данных
- SheetJS (xlsx) — экспорт в Excel
- Python — CORS прокси сервер
- Чистый HTML/JS, без фреймворков
